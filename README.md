# Place Anywhere Core

自由方块放置核心系统 - 支持任意位置/任意旋转放置方块

## 支持版本

| 平台 | 1.18.2 | 1.19.2 | 1.19.4 | 1.20.1 | 1.20.6 | 1.21.1 |
|------|--------|--------|--------|--------|--------|--------|
| Fabric | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ |
| Forge | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ |

## 分支结构

- `main` - 说明文档 + Release 链接
- `fabric-1.18.2` ~ `fabric-1.21.1` - Fabric 各版本源码
- `forge-1.18.2` ~ `forge-1.21.1` - Forge/NeoForge 各版本源码

> **目前仅 `fabric-1.21.1` 为活跃开发分支，其他版本暂时不会同步更新。**
> 待 1.21.1 Fabric 版本功能完善后，再同步更新其他版本。

## 下载

前往 [Releases](https://github.com/gutianhuan0/place-anywhere-core/releases) 下载对应版本的 JAR 文件。

## 安装

1. 下载对应 Minecraft 版本和加载器的 JAR 文件
2. 将 JAR 放入 `mods` 文件夹
3. 需要同时安装 Place Anywhere Core（如果使用 Free Placement）

## 依赖关系

- **Place Anywhere Core**（必需）- 自由方块核心系统
- **Place Anywhere Free Placement**（可选）- 自由放置模式
- **Better Slab**（独立）- 合并半砖+竖直半砖
